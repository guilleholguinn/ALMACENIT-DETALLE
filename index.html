Preguntas

¡DESPEJÁ TUS DUDAS!

¿Este curso es adecuado para principiantes en fitness?

¡Sí, el curso Top100 está diseñado para adaptarse a todos los niveles de condición física. Nuestros instructores están capacitados para ajustar los ejercicios según tus necesidades y nivel de experiencia, asegurando que puedas participar de manera segura y efectiva.

¿Cómo se programan las sesiones de entrenamiento?

Las sesiones de entrenamientO están programadas de manera flexible para adaptarse a diferentes horarios y rutinas. Puedes elegir entre varias opciones de horarios que mejor se ajusten a tu agenda diaria.

¿Puedo recuperar clases en caso de cancelar?

Generalmente, nuestro centro ofrece opciones flexibles para recuperar clases en caso de cancelación. Comunícate con nuestro equipo de atención al cliente para discutir los detalles específicos y las políticas aplicables a la recuperación de clases. Estamos aquí para asegurarnos de que aproveches al máximo tu experiencia con nosotros.

¿Qué tipo de actividades están incluidas en el curso Top100?

El curso Top100 incluye una amplia variedad de actividades, como entrenamiento de fuerza, cardio intenso, yoga para la flexibilidad y clases de spinning. Esto asegura una experiencia integral que abarca diferentes aspectos del fitness y el bienestar.

¿Hay algún requisito especial de equipo o vestimenta para participar?

Recomendamos usar ropa deportiva cómoda y calzado adecuado para ejercicio. Si tienes algún equipo específico necesario para ciertas actividades (como una esterilla de yoga o zapatos de spinning), te informaremos con anticipación.

¿Cuanto dura cada clase?

El curso Top100 tiene una duración flexible que puede variar según el plan que elijas. Las sesiones suelen durar entre 60 a 90 minutos, dependiendo del tipo de actividad programada para ese día. La estructura de las sesiones varía para ofrecer una experiencia dinámica y efectiva. .
